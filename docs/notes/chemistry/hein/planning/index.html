LLMs

  • rule based ‚Üí statistical based ‚Üí neural network-based models

  • RNN and LSTM ‚Üí vanishing gradients ‚Üí loss of very long sequence contexts

  • transformer model: self-attention (SA) mechanisms ‚Üí focus on different parts of long input in parallel

    • positional encodings: maintain awareness of word/token order

    • self-attention: query, key, value system ‚Üí contextualize dependencies ‚Üí softmax

  • CLM: gpt-4: text generation is sequential and dependent on preceding context ‚Üí predict next word based on prev word statistically

  • MLM: BERT understand directional context, learn forward and backward language dependencies

  • Seq2Seq model: transform input into related output ‚Üí translation or summarization
