t ::=         Terms
          x   Variable
        | 位x. t Abstraction
        | t t Application
  • (位x. t) is a function with formal parameter x that returns the term t
  • 位x.位y x) is a function that takes two arguments x and y and returns the first argument
  • (t_1 t_2) is a function call where t_1 is a function and t_2 is the supplied argument
  • ((位x. x) y) supplies y as the argument to the identity function

Parentheses can be dropped:

  • application is left associative ((f f) x) is the same as f f x

  • 位 binds as much as possible to the right: 位f. 位x. f (f x) is the same as (位f. (位x. f (f x)))

  • multiple consecutive abstractions can be combined: 位f.位x.f (f x) is same as 位f x. f (f x)

Example of Application

  1. ((位wyx. y (w y x)) (位sz. z))
  2. ((位yx. y ((位sz. z) y x)))
  3. (位yx. y ((位sz. z) y x))
  4. (位yx. y x)

Another way:

  1. ((位wyx. y (w y x)) (位sz. z))
  2. (位yx. y ((位sz. z) y x))
  3. (位yx. y ((位z. z) x))
  4. (位yx. y x)
  • an application: replace every occurrence of the formal parameter in the body of the function with the given argument

Encoding Booleans in the 位-Calculus: The Church Encoding of Booleans

  • true: 位x. 位y. x
  • false: 位x. 位y. y
  • &&: 位x. 位y. ((x y) false)
  • ||: 位x. 位y (( x true) y)
  • !: 位x. ((x false) true)
  • if: 位c. 位t. 位e. ((c t) e)

Example:

  1. (true && false)
  2. (位x y. ((x y) false))
  3. (位y. (((位x. 位y. x) y) false)) apply x as true
  4. (((位x. 位y. x) (位x. 位y. y)) false) apply y as false
  5. (((位xy. x) (位xy. y)) false)
  6. ((位y. (位xy. y)) false) outer y is false but dont apply it anywhere
  7. (位xy. y)
  8. false

Church Encoding of Natural Numbers in the 位-Calculus

  • 0: 位s. 位z. z
  • 1: 位s. 位z. (s z)
  • 3: 位s. 位z. (s (s z))
